## Homebrew Snippy

This repositories holds the forumla file for [snippy](https://github.com/MohamedElashri/snipo/tui), a TUI client for [Snipo](https://github.com/MohamedElashri/snipo/)



If you use Homebrew, you can install Snippy directly through the tap:

```bash
brew install MohamedElashri/snippy/snippy
```
