## Homebrew Snippy

This respoitory holds the forumla file for [snippy](https://github.com/MohamedElashri/snipo/tui), a TUI client for [Snipo](https://github.com/MohamedElashri/snipo/)



### Installation

You can install Snippy directly through the tap:

```bash
brew install MohamedElashri/snippy/snippy
```
